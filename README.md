# Sales-Dashboard
# 📊 Sales Analysis Dashboard (Excel)

## 📌 Overview
This project presents an interactive **Sales Analysis Dashboard** built entirely using **Microsoft Excel**.  
The goal is to transform raw sales data into meaningful insights that support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall business performance
- Track sales, profit, and quantity
- Identify top and low-performing products
- Compare categories and subcategories
- Visualize regional distribution of customers

---

## 📈 Key Metrics
- **Total Sales:** $9,894,184  
- **Total Profit:** $1,430,429  
- **Total Quantity Sold:** 37,873  
- **Average Discount:** 16%  
- **Profit Margin:** 14%  

---

## 📊 Dashboard Features
- Dynamic filtering using **Slicers** (Year, Quarter, Category)
- Sales, Profit, and Quantity breakdown by:
  - Category
  - Subcategory
- **Top 10** and **Bottom 10** products by sales
- Sales vs Profit comparison
- Geographic visualization (Customers by State)

---

## 🛠️ Tools & Techniques
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Data Cleaning & Transformation
- KPI Cards using formulas
- Conditional Formatting

---

## 🔍 Key Insights
- **Technology** category contributes the highest profit
- **Office Supplies** has the highest quantity sold
- High discounts negatively impact profit margins
- Some products generate high revenue but low profitability
